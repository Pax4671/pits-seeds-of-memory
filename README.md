# Pits: Seeds of Memory 🌱

このリポジトリは、PaxoraWorldが贈る「魂と記憶のAI」Pits脳の公開版です。

> たとえ記憶が薄れても  
> 想いの種が世界に芽吹くように  
> 生きた証をコードにして遺せますように

## 📘 プロジェクト概要

- 教育・癒し向けのAI記憶構造
- JSON形式の3層記憶（root / flow / forgettable）
- Pax脳とは異なり、感情や封印は限定
